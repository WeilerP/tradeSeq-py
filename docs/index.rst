Welcome to tradeSeq-py's documentation!
=======================================

General
-------
This repository implements the R package tradeSeq in Python. The code base is still experiencing heavy development.
While we reimplement the tradeSeq package, exact reproducibility is not guaranteed. Differences in result are
caused by relying on Python packages instead of their R counterparts in some cases.
