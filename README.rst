Environment setup
^^^^^^^^^^^^^^^^^

To easily rerun notebooks withouth having to change the kernel, run

```bash
conda create -n py-tradeSeq-py39 python=3.9 --yes && conda activate py-tradeSeq-py39
```

Installation
^^^^^^^^^^^^

```bash
pip install -e ".[dev]"
```
